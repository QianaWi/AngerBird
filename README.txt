在背景主题添加后，由于层级问题，拖尾和弹弓的线都看不到，这时候有两种解决方法
第一种：将主题背景的图片层级改为default，order改为-1；
第二种：由于是2D游戏，可以将主题背景图片的Z轴设置成负数，如-0.1
